== Installation ==

1. Upload the `wp-ai-assistant` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to WP AI Assistant > Settings
4. Enter your API Key and configure the chat widget
5. The chat widget will automatically appear on your website

== Frequently Asked Questions ==

= Do I need an API key? =

Yes, you need an API key from the WordPress Assistant platform. Sign up at https://ai.webtamino.com to get your API key.

= Does it work with WooCommerce? =

Yes! The plugin has deep integration with WooCommerce for product search, recommendations, and order creation.

= Is it mobile responsive? =

Absolutely! The chat widget is fully responsive and works beautifully on all devices.

= Can I customize the appearance? =

Yes, you can customize colors, position, greeting messages, and more from the settings page.

= Does it support Persian/Farsi? =

Yes! The plugin has full bilingual support with RTL for Persian/Farsi.

= Will it slow down my website? =

No, the plugin is highly optimized and has minimal impact on page load times (<50ms).
